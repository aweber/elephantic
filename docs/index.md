# Elephantic

An ORMish library based on Pydantic for building and executing SQL queries.

## Features

- **Modern Python**: Requires Python 3.12+ with full type hint support
- **Pydantic Integration**: Built on Pydantic for robust data validation and serialization
- **Type-Safe**: Leverages Python's type system for safe database operations
- **Query Builder**: Intuitive API for constructing SQL queries
- **Database Agnostic**: Works with multiple database backends

## Why Elephantic?

Elephantic combines the flexibility of raw SQL with the safety and convenience of Pydantic models. It provides:

- Type-safe query construction
- Automatic validation of query parameters
- Clear, readable query building syntax
- Integration with modern Python async patterns
